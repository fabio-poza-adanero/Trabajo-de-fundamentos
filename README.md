# Trabajo individual - Fundamentos de la Ingeniería Informática
(he usado un poco de inteligencia artificial para perfeccionar la web y corregir errores)
## Estructura de la web
La web está formada por varias páginas:
- `index.html`: Página principal. Presento brevemente la web y explico qué se puede encontrar en cada sección.
- `about.html`: Página "Sobre mí". Incluye una pequeña descripción personal, mi CV básico, una imagen y un enlace a mi GitHub.
- `contact.html`: Página de contacto. Tiene un formulario simple con nombre, correo, asunto y mensaje. El formulario no envía nada, solo es de ejemplo.
- `net.html`: Página de red. Aquí están los enlaces a las webs de algunos compañeros.
- `topic.html`: Página temática. He elegido como tema la **Inteligencia Artificial aplicada a la informática**.
- `degree.html`: Página sobre el grado. Explico de forma general el **Grado en Ingeniería Informática** y pongo las asignaturas de primero (primer y segundo cuatrimestre) con una breve descripción.
- `fii.html`: Página de la asignatura **Fundamentos de la Ingeniería Informática**, con una explicación más detallada, objetivos, contenidos y lo que se espera que aprendamos.
Además:
- Carpeta `css/` con la hoja de estilos `style.css`.
- Carpeta `img/` donde está la imagen que utilizo en `about.html`.
## Problemas y cómo los he solucionado
Durante el desarrollo me he encontrado con algunos problemas típicos:- **Tamaño de la imagen en "Sobre mí"**: Al principio la foto salía demasiado grande. Lo he solucionado poniendo un `max-width` en el CSS para limitar el ancho de las imágenes.
- **Rutas de archivos**: He tenido que revisar bien las rutas de `css/style.css` y de la imagen en `img/` para que se cargaran correctamente en todas las páginas.
- **Menú de navegación**: He comprobado que todos los enlaces del menú apunten al archivo correcto y que el menú sea igual en todas las páginas.
## Conclusiones
Con este trabajo he aprendido a:
- Crear una web sencilla usando solo HTML y CSS.
- Organizar mejor los archivos de un proyecto (páginas, estilos, imágenes y README).
- Entender cómo publicar una página en GitHub Pages a partir de un repositorio.
